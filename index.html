How does it work?

Current version: 1.0

This program is very simple. You input a prequel quote (or a part of it) and the program answers with an appropriate response. The program doesn't have any sort of user interface, so it opens in Windows CMD or console.

Currently it features quotes from The Phantom Menace, Attack of the Clones and Revenge of the Sith. In the future I'm planning to add quotes from the likes of Clone Wars, Rogue One, prequel video games etc.
